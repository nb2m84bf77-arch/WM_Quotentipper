Quote → Prob → Erwartungswert

Tippquote Heatmap

Quoten für Einzelergebnisse eintragen, Punkteschema definieren — die Matrix zeigt den Erwartungswert pro möglichem Tipp.

Punkteschema

K.o.-Phase / Elfmeter

Unentschieden-Wahrscheinlichkeiten (0:0, 1:1, 2:2 …) werden auf Sieg A / Sieg B verteilt — nicht starr 50/50, sondern geblendet: zur Hälfte nach der ursprünglichen 90-Min-Sieg-Tendenz (aus den Nicht-Unentschieden-Ergebnissen), zur Hälfte neutral 50/50 (Verlängerung + Elfer gleichen Stärkeunterschiede etwas aus). Davon je 75% auf Tordifferenz ±1, 25% auf ±2 (kurze 30-Min-Verlängerung + Elferschießen ergeben meist nur 1 Tor Unterschied). Höhere Differenzen werden ignoriert. Die Punktevergabe nutzt dabei dein reguläres Schema oben (Tendenz / Tendenz+Tordifferenz).

Hochrechnung fehlender Quoten

Leere Felder am Rand der Matrix werden aus dem Trend der vorhandenen Werte je Zeile/Spalte fortgeschrieben (linear: konstante Differenz, exponentiell: konstanter Faktor). Mind. 2 Werte pro Reihe nötig.

Daten

Quoten — Ergebnis genau X:Y

Erwartete Punkte je Tipp

0%-Perzentil (schwach)
100%-Perzentil (stark) · Gelb bei 80%

Rechenweg

Alles was die Heatmap oben berechnet, in vier Schritten — falls du nachvollziehen willst was unter der Haube passiert oder die Logik selbst überprüfen möchtest.

01

Quoten → Quotienten

Für jedes Ergebnis X:Y wird die Quote ins Verhältnis zur Quote von 0:0 gesetzt: quotient(X:Y) = quote(0:0) / quote(X:Y). Eine doppelt so niedrige Quote bedeutet doppelt so wahrscheinlich, daher ist dieses Verhältnis direkt proportional zur tatsächlichen Wahrscheinlichkeit.

02

Quotienten → Wahrscheinlichkeiten

Alle Quotienten der Matrix werden aufsummiert. Jeder einzelne Quotient wird durch diese Summe geteilt — dadurch ergeben alle Wahrscheinlichkeiten zusammen exakt 100%. Dieser Normierungsschritt entfernt automatisch die Wettanbieter-Marge (Overround), ohne dass man sie separat herausrechnen muss.

03

Wahrscheinlichkeiten → Erwartungswert

Für jeden möglichen Tipp A:B wird über alle Ergebnisse aus der Matrix summiert: EV(Tipp) = Σ P(Ergebnis) × Punkte(Tipp, Ergebnis). Die Punktezahl pro Kombination ergibt sich aus deinem Punkteschema (exakt, Tendenz+Tordifferenz, Tendenz, Unentschieden-Tendenz, falsch).

04

K.o.-Phase: Verlängerung & Elferschießen

Ist die K.o.-Option aktiv, kann kein 90-Minuten-Unentschieden als Endergebnis stehen bleiben — die gesamte Unentschieden-Wahrscheinlichkeitsmasse (Summe aller P(0:0), P(1:1), P(2:2) …) wird umverteilt:

Sieg-Tendenz: Aus der Matrix wird die reine 90-Minuten-Sieg-Verteilung ermittelt — Summe aller P(Sieg A) zu Summe aller P(Sieg B), ohne die Unentschieden. Diese Tendenz wird zur Hälfte beibehalten, zur Hälfte Richtung neutralem 50/50 gezogen (Verlängerung + Elfer gleichen Stärkeunterschiede teilweise aus): geblendet(A) = 0,5 × Tendenz(A) + 0,5 × 0,5.

Tordifferenz im Elfer: Innerhalb des jeweiligen Team-Anteils wird die Wahrscheinlichkeitsmasse auf Tordifferenz ±1 und ±2 aufgeteilt, im Verhältnis 3:1 (75% / 25%). Diese Annahme berücksichtigt, dass sowohl die kurze 30-minütige Verlängerung als auch das Elferschießen meist nur eine knappe Differenz von einem Tor erzeugen. Höhere Differenzen (±3 und mehr) werden als zu unwahrscheinlich vernachlässigt.

Vereinfachung: die 2:1-Aufteilung der Tordifferenz ist unabhängig von der Sieg-Tendenz — sie kommt rein aus der Elfer-Mechanik, nicht aus dem Stärkeverhältnis der Teams.
